## Goals 

1. Gatsby 개발 환경 구축

2. Hero 섹션에 Background Video 삽입 및 반응형 폰트 구축

3. Comment 섹션에 Material UI 삽입, Display grid 사용

4. 각 섹션 데이터들은 data 폴더에 따로 보관
